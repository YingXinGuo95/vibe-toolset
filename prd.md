# 备忘录工具 PRD

## 一、产品定位
一个极简的本地优先备忘录工具，支持按天记录事件、状态流转、快速回顾。

---

## 二、核心功能模块

### 2.1 事件记录（核心）

| 功能点 | 说明 |
|--------|------|
| 新建事件 | 输入框 + 回车/按钮提交，自动记录创建时间 |
| 状态流转 | 待完成 → 已完成（点击勾选）/ 废弃（点击废弃按钮） |
| 删除 | 永久删除，需二次确认（或支持撤销） |
| 编辑 | 点击事件文本可编辑，失焦或回车保存 |

**状态定义：**
- `待完成`（默认）：灰色或正常显示
- `已完成`：文字划线 + 绿色勾选标记
- `废弃`：文字划线 + 灰色/红色标记，可恢复

### 2.2 按天归类展示
- 以**天**为单位分组展示事件
- 每天是一个卡片/区块，顶部显示日期（如"6月16日 周一"）
- 支持展开/折叠某天（默认展开今天，折叠历史天）
- 空天可隐藏或显示"暂无事件"占位

### 2.3 时间范围筛选（回顾功能）
- 顶部提供**日期范围选择器**（如最近7天、最近30天、自定义范围）
- 筛选后只展示该范围内的天数卡片
- 统计信息：范围内总事件数、已完成数、完成率

## 三、页面结构与交互

### 3.1 整体布局

```

│  [Logo] 备忘录          [日期筛选 ▼] │  ← Header
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐   │
│  │  📝 今天有什么要做的？       │   │  ← 输入区（吸顶或固定顶部）
│  │  [输入框...........] [添加]  │   │

├─────────────────────────────────────┤

│  │ ☐ 买牛奶                    │   │  ← 待完成

├─────────────────────────────────────┤

```

### 3.2 事件卡片交互细节

**待完成事件：**
- 左侧圆形复选框，点击后变为已勾选 + 文字划线动画
- 右侧悬浮操作：编辑（✏️）、废弃（🗑️）、删除（❌）

**已完成事件：**
- 左侧勾选图标（绿色），文字中划线
- 显示完成时间（"✓ 14:32"）
- 点击可取消完成（恢复待完成）

**废弃事件：**
- 文字灰色划线，显示"已废弃"标签
- 操作：恢复（♻️）、删除（❌）

**编辑模式：**
- 点击文本进入编辑，输入框自动聚焦
- 回车或失焦保存，ESC取消

### 3.3 日期筛选器
- 快捷选项：今天 / 最近7天 / 最近30天 / 本月 / 全部
- 自定义：日期范围选择器（shadcn Calendar + Popover）
- 筛选后页面标题动态变化（如"最近7天的事件"）

## 四、数据结构

```typescript
interface MemoEvent {
  id: string;           // uuid
  content: string;      // 事件内容
  status: 'todo' | 'done' | 'abandoned';
  createdAt: number;    // 时间戳
  completedAt?: number; // 完成时间
  updatedAt: number;
}

// 按天分组展示时的结构
interface DayGroup {
  date: string;         // "2026-06-16"
```

## 六、视觉风格建议
- **极简干净**：白底 + 灰色边框，重点用色克制
- **状态色**：
  - 待完成：黑色文字
  - 已完成：绿色勾选 + 灰色划线文字
  - 废弃：灰色文字 + 红色"废弃"标签
- **动画**：状态切换时有轻微过渡动画（0.2s），提升质感
- **空状态**：无事件时显示插画/图标 + "今天还没有记录哦"

---

## 七、MVP 边界（明确不做）
- ❌ 用户登录/云端同步（纯本地）
- ❌ 提醒/通知功能
- ❌ 富文本编辑（纯文本即可）
- ❌ 标签/分类系统（后续可扩展）
- ❌ 重复事件

---
